Universal Trade Platform — Documentation

Unified multi-broker trading API (FastAPI) supporting Robinhood, E*TRADE, and IBKR. Features daemon mode, execution store, trade simulation, conId-based deduplication, and market data streaming.

Quick Start

# Start daemon (recommended)
python utp.py daemon --live

# Run all tests (359 tests)
python -m pytest tests/ -v

# CLI auto-detects daemon
python utp.py portfolio
python utp.py quote SPX NDX
python utp.py executions --live

# Execute a playbook (paper)
python utp.py playbook execute playbooks/example_mixed.yaml --paper
